There are a lot of odd entities on the internet, one of which is the animated gif. After completing yourimage.io, I was interested in extending the idea of random image collage and image painting to make use of animated content.

Utilizing gif.js, native canvas api and imgur I was able to create a website that has the ability to paint and collage with animated content.
