I used the JavaScript canvas API, paper.js and the Imgur API to create this website. It allows users to use their mouse or touch device to paint with images from the Imgur API. The program also allows users to save their creation. Using the combination of the unknown (Not knowing which image will be next), collage (https://en.wikipedia.org/wiki/Collage), and a version of action painting (https://en.wikipedia.org/wiki/Action_painting) the results become unique artwork to share. The generative nature of this website allows a collaboration between the user and the computer. Using pixels to show weight and density while expressing unique ideas with juxtapostion of textual and visual ideas from Imgur's site.
